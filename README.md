

# 🐍 Python-Lernreise – 30 Tage Python

Willkommen in meinem Repository!

In diesem Repository dokumentiere ich meine **30-tägige Reise, Python von Grund auf zu lernen**. Jeden Tag bearbeite ich neue Themen, programmiere Übungen und Mini-Projekte und halte meinen Lernfortschritt fest.

Mein Ziel ist es, nicht nur Python zu lernen, sondern gleichzeitig ein aussagekräftiges GitHub-Portfolio aufzubauen, das meine Entwicklung als Programmierer zeigt.

---

## 🎯 Ziele

* Python von Grund auf lernen

* Git und GitHub sicher anwenden
* Sauberen und verständlichen Code schreiben
* Meinen Lernfortschritt dokumentieren
* Ein Portfolio für Bewerbungen und zukünftige Projekte erstellen
